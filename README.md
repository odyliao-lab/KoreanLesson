# KoreanLesson

「별빛韓語研究所｜Starlight Korean Lab」是為國中生設計的互動韓語學習網站，以 iPad 橫向操作為主要介面，並支援電腦瀏覽器。

正式網站：[korean.odyliao.cc](https://korean.odyliao.cc)

## 課程內容

- 初級：20 天零基礎課程
- 中級「舞台準備中」：15 天情境與文法課程
- 高級「正式出道」：15 天完整表達與正式發表課程
- 每次學習約 45–60 分鐘，包含教學、練習、韓語語音與自習任務
- 每日八種互動題型、每五天涵蓋前五課的 10 題累積檢核
- 50 課固定韓語語音檔，正常與慢速播放不依賴裝置語音
- 五位原創男性學習嚮導的個人角色圖與課堂狀態
- 錯題複習與成果評分表
- 每題分段提示、答案解析、常見錯法與家長／老師協助建議
- 瀏覽器內發音錄音、立即回放與三星自評；錄音不會上傳或同步
- 依個別題目、錯誤次數與提示使用量排序的精準錯題複習
- 免登入本機模式、選擇性登入、跨裝置雲端同步與版本衝突保護
- 學習時間、連續天數、間隔複習、成就徽章與資料匯入匯出
- 教師班級、邀請碼、課程指派完成率、學生錯題診斷與 CSV 學習報告
- 作業分數、遲交、逾期、重做次數與班級平均趨勢
- 班級成員／作業管理、邀請碼更新，以及家長與特定學生關聯
- 班級題目回報中心：學生可將疑問題送交老師追蹤處理
- 可分享至特定階段、Day、學習階段與題號的接續網址
- PWA 離線支援、高對比模式、鍵盤導覽與隱私說明

## 本機開發

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

建置驗證：

```bash
npm run build
```

## 技術

- React / Next.js 相容應用架構
- vinext + Vite
- OpenAI Sites / Cloudflare runtime
