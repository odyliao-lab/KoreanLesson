# KoreanLesson

「별빛韓語研究所｜Starlight Korean Lab」是為國中生設計的互動韓語學習網站，以 iPad 橫向操作為主要介面，並支援電腦瀏覽器。

正式網站：[korean.odyliao.cc](https://korean.odyliao.cc)

## 課程內容

- 初級：20 天零基礎課程
- 中級「舞台準備中」：15 天情境與文法課程
- 每次學習約 45–60 分鐘，包含教學、練習、韓語語音與自習任務
- 使用瀏覽器本機儲存學習進度與筆記，第一版免登入

## 本機開發

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

建置驗證：

```bash
npm run build
```

## 技術

- React / Next.js 相容應用架構
- vinext + Vite
- OpenAI Sites / Cloudflare runtime
